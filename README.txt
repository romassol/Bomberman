Игра "Бомбермен"

Версия 0.7
Автор: Ромасс Ольга

Описание
Данное приложение является реализацией игры "Бомбермен".

Требования
Python версии не ниже 3.5
PyQt версии 5

Состав
Графическая версия: main.py
Изображения: images/
Тесты:  test_game.py
	test_point.py
	test_reader.py

Графическая версия
Справка по запуску: python main.py --help
Пример запуска: python main.py

Подробности реализации
В основе всего лежит класс Game, реализующий хранение персонажей, передвижение их и взаимодействие одних персонажей с другими (взрыв бомбы, соприкосновение bm и монстров). Класс Reader обрабатывает данные из json-формата.
